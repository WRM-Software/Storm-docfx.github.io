    Storm Web API Documentation

    Welcome to the Storm Web API documentation. This documentation covers the various services and APIs available in the Storm platform.

    Overview

    Storm is a comprehensive platform for managing workforce operations, including:

    • Callouts: Managing emergency response and resource allocation
    • Hotel Assignments: Managing crew accommodation and lodging
    • Time Sheets: Tracking work hours and scheduling
    • Resources: Managing personnel and equipment
    • Events: Managing operational events and incidents

    Architecture

    The Storm platform consists of several key components:

    • Storm.Biz: Core business logic and services
    • Storm.Facing: gRPC API services
    • Storm.RealTimeUpdate: Real-time notification system
    • Storm.RealTimeTunnel: Real-time data tunneling service
